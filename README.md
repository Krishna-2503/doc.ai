# 🤖 DocAssist AI

DocAssist AI is a modern AI-powered web application that helps users understand government documents through simple, accurate, and easy-to-read explanations. It provides a clean, responsive interface where users can ask questions and receive AI-generated answers instantly.

---

## 🚀 Features

- 🤖 AI-powered document assistance
- 📄 Government document explanations
- ⚡ Fast and accurate responses
- 🎨 Modern and responsive UI
- 📱 Mobile-friendly design
- 🔍 Easy search interface
- ✨ Smooth hover effects and animations

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- AI Agent
